ABSTRACT:

Numa primeira fase pretendeu-se analisar o dataset das pubs do jcr, para isso com o auxílio do editor Oxygen XML Editor gerei um DTD de forma aperceber a estrutura deste dataset.

De seguida foi usado o Protégé para construir a parte estrutural da ontologia e criar alguns indivíduos para servirem de modelo à próxima fase.

Posteriormente, criei uma script em Python para povoar a ontologia com os indivíduos do dataset, após gerado o ficheiro de texto acrescentei à ontologia que serviu de base os indivíduos e inferi com o Reasoner disponibilizado mais alguma informação de forma a tornar a ontologia o mais completa possível.

Em seguida carreguei a ontologia no Ontobud, para isso criou-se um repositório de nome A83732-TP5 e importou-se o ficheiro inferido gerado previamente.

Por fim, criou-se uma API de dados com as funcionalidades de CRUD sobre o modelo criado, isto é, inclui as seguintes rotas: GET /pubs , GET /pubs?type= , GET /pubs/{id} , GET /authors e GET /authors/{id} .
