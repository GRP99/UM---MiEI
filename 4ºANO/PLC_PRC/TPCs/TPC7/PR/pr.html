ABSTRACT:

No presente trabalho foi usado o software Protégé para criar a parte estrutural da ontologia sobre um dataset disponibilizado acerca de cidades e suas ligações.

De seguida com recurso a um script em Python efetuou-se o processamento do dataset gerando um ficheiro dos indivíduos presentes na sintaxe TTL, que posteriormente foi adicionada à ontologia base criada.
