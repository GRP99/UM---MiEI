ABSTRACT:

No presente trabalho pretendeu-se desenvolver uma aplicação Web com Vue.js para consumir uma API de dados sobre as publicações do professor JCR.

Partindo da API de dados previamente construída que fornece algumas funcionalidades de CRUD sobre uma ontologia das publicações armazenada na plataforma Ontobud, apenas criei uma aplicação com a framework Vue.js.

Nesta aplicação é possível visualizar a lista de publicações, bem como a lista de autores que contribuiram para as mesmas. Em cada elemento da lista é possível aceder à página individual da entidade em causa.
